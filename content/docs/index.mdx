BrushBots，中文名为小刷子，是一款专注于自动化和优化空投执行的工具。最初，我们开发这款工具是为了满足自己的需求，以便更高效地完成一些空投任务。经过六个月的内部使用和不断完善，BrushBots 已经演变成一个具有完全可视化 UI 的产品，并且在实际应用中获得了一定的空投收益。现在，我们准备将这款产品介绍给更广泛的 Web3 用户群体。

BrushBots 集成了多个 AI 模型，包括 OpenAI 和 Model Proxy，使我们能够在特定场景中充分利用 AI 的输出。我们还增加了多项常用功能，例如 Token 的归集、分发，以及从中心化交易所（CEX）的批量提现。由于我们主要面向中国用户，BrushBots 特别集成了来自中国顶尖 KOL 社区内部群的消息，确保用户能够及时获取到相关信息。

BrushBots 基于 Electron 和 Node.js 技术栈构建，包含了广泛的底层 API，例如 Twitter 操作、数据存储、API 调用管理、代理等。这一强大的基础使得我们能够快速开发和实现特定项目的脚本，大幅提高开发速度。一旦确定了项目和空投策略，我们可以迅速完成所需的编码和执行。

我们的架构设计既合理又灵活，这要归功于团队成员在中国大型互联网公司的丰富经验。从一开始，BrushBots 就以整体架构为核心，抽象了钱包和账号等多层次的数据结构，确保了产品的稳定性和可扩展性，使其成为 Web3 用户的可靠工具。
